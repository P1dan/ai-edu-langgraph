## 阶段一：定义筑基与关系建模（第1–4周｜总54小时｜理论16h + 实践38h）

### 模块1.1｜任意角与单位圆定义的几何统一  
- **学习主题**：从初中锐角三角函数到任意角的自然延拓  
- **学习目标**：能用单位圆坐标精确解释 sinα、cosα、tanα 的符号、绝对值与几何意义；能对比直角三角形定义在第二、三、四象限的失效原因，并说明单位圆如何修复该局限  
- **学习内容**  
  - 角的推广：正角/负角/零角/象限角/轴线角的旋转定义  
  - 单位圆定义：P(x,y) = (cosα, sinα)，tanα = y/x（x≠0）的坐标本质  
  - 三种表示统一：直角三角形（仅限第一象限）、坐标比值（全象限）、有向线段（正弦线/余弦线/正切线）  
  - 符号判定口诀“一全正，二正弦，三正切，四余弦”的单位圆路径推演（手绘6类诱导路径图谱）  
- **前置依赖**：初中直角三角形三角函数定义、平面直角坐标系、弧度制概念（需复习π rad = 180°换算）  
- **建议学习时间**：第1周·3.5小时（含2h动态手绘+1.5h对比反思笔记）

### 模块1.2｜同角三角函数基本关系式网络构建  
- **学习主题**：从定义出发推导并结构化记忆三大关系式及其衍生链  
- **学习目标**：能从单位圆定义独立推导平方关系、商数关系、倒数关系；能双向使用（化简↔证明），并在含参表达式中自动识别可约简结构（如 1−sin²x → cos²x）  
- **学习内容**  
  - 核心三式：sin²α + cos²α = 1；tanα = sinα/cosα；1 + tan²α = sec²α（及余割、余切对应式）  
  - 推导逻辑链：单位圆 x²+y²=1 → sin²+cos²=1 → 两边同除cos² → 1+tan²=sec²  
  - 常见变形陷阱：√(1−cos²x) ≠ sinx（必须加绝对值或讨论象限）；tanα·cosα = sinα（隐含cosα≠0）  
  - “关系式选择决策树”：给定sinα求cosα → 先判象限 → 再定符号 → 最后开方  
- **前置依赖**：模块1.1全部内容；平方根与绝对值运算规则  
- **建议学习时间**：第1周·3小时（含1h推导实操+2h错例辨析训练）

### 模块1.3｜诱导公式：单位圆上的对称运动建模  
- **学习主题**：将9组诱导公式还原为单位圆上4种刚体变换（对称/旋转）  
- **学习目标**：能不依赖口诀，通过单位圆上点P(cosα,sinα)经轴对称（x轴/y轴/原点）或π±α旋转，直接写出对应角的三角函数值；能解释“奇变偶不变，符号看象限”的几何本源  
- **学习内容**  
  - 四大变换映射表：  
    ▪️ 关于x轴对称 → α → −α → (cosα, −sinα) → sin(−α)=−sinα  
    ▪️ 关于y轴对称 → α → π−α → (−cosα, sinα) → cos(π−α)=−cosα  
    ▪️ 关于原点对称 → α → π+α → (−cosα, −sinα)  
    ▪️ 绕原点旋转π/2 → α → π/2−α → (sinα, cosα) → sin(π/2−α)=cosα  
  - 9组公式归类：k·π/2 ± α型（k=0,1,2,3,4）的周期性与奇偶性分类  
  - 实战检验法：“cos(7π/6)” → 定位7π/6在第三象限 → 看作π+π/6 → 用π+α公式 → −cos(π/6) = −√3/2  
- **前置依赖**：模块1.1（单位圆坐标理解）、模块1.2（符号判定能力）  
- **建议学习时间**：第2周·4小时（含2h变换手绘+2h真题反向溯源训练）

### 模块1.4｜和差角公式的几何推导与双向应用  
- **学习主题**：从单位圆上两点距离公式出发，严格推导cos(α−β)，再导出其余公式  
- **学习目标**：能独立完成cos(α−β)的单位圆向量推导；能由cos(α−β)推出cos(α+β)、sin(α±β)、tan(α±β)；能在恒等变形中识别“角拆分”时机（如sin75°=sin(45°+30°)）  
- **学习内容**  
  - 核心推导：A(cosα,sinα), B(cosβ,sinβ) → |AB|² = 2−2cos(α−β) = (cosα−cosβ)²+(sinα−sinβ)² → 展开整理得cos(α−β)  
  - 公式链生成：cos(α+β)=cos[α−(−β)] → sin(α±β)=cos(π/2∓(α±β)) → tan=(sin)/(cos)  
  - 应用模式识别：  
    ▪️ “已知sinα,cosβ,求sin(α+β)” → 直接代入  
    ▪️ “化简sinxcosy+cosxsiny” → 识别为sin(x+y)  
    ▪️ “证明tanα+tanβ=tan(α+β)(1−tanαtanβ)” → 从右向左展开验证  
- **前置依赖**：模块1.1–1.3；两点间距离公式；代数恒等变形能力  
- **建议学习时间**：第2–3周·5小时（含2.5h推导实操+2.5h模式识别专项训练）

### 模块1.5｜倍角、半角与积化和差公式的结构化整合  
- **学习主题**：以和差公式为母体，生成倍角/半角/积化和差三级公式体系  
- **学习目标**：能从sin(α+β)推导sin2α、cos2α三形式、tan2α；能由cos2α反解cosα → 推出半角公式；能将sinαcosβ快速转为和差；能根据题目目标（化简/求值/降幂）自主选择公式层级  
- **学习内容**  
  - 倍角公式：sin2α=2sinαcosα；cos2α=cos²α−sin²α=2cos²α−1=1−2sin²α；tan2α=2tanα/(1−tan²α)  
  - 半角公式：cos²(α/2)=(1+cosα)/2 → cos(α/2)=±√[(1+cosα)/2]（符号由α/2所在象限定）  
  - 积化和差：sinαcosβ=[sin(α+β)+sin(α−β)]/2（4组均需掌握）  
  - 公式选用决策指南：  
    ▪️ 含sin²x/cos²x → 优先cos2α降幂  
    ▪️ 含√(1−cosα) → 立即启动半角公式并判符号  
    ▪️ 含sin3x+sinx → 用和差化积sinA+sinB=2sin((A+B)/2)cos((A−B)/2)  
- **前置依赖**：模块1.4全部内容；开方运算与象限判断  
- **建议学习时间**：第3–4周·6小时（含3h推导链构建+3h“公式靶向选择”限时训练）

### 模块1.6｜阶段一闭环验证：定义一致性监控力认证  
- **学习主题**：通过四项强制输出，完成阶段一能力微认证  
- **学习目标**：达成“能向虚拟学生清晰解释几何本质”的教学级理解  
- **学习内容**

  - ✅ 手写《公式双向推导链》：同角关系（定义→平方关系→商数关系）、和差→倍角→半角→积化和差，每步标注几何或代数依据  
  - ✅ 口语化讲解录音（≥3分钟）：针对“为什么cos(π−α)=−cosα”，用单位圆坐标+对称性+初中学过的余弦定义三重解释

- **前置依赖**：模块1.1–1.5全部完成  
- **建议学习时间**：第4周末·3小时（集中完成四项输出，作为阶段一结业凭证）

## 阶段二：图象解码与参数化思维（第5–8周｜总54小时｜理论16h + 实践38h）

### 模块2.1｜正弦/余弦/正切函数的基本图象与性质  
- **学习主题**：从定义出发绘制y=sinx, y=cosx, y=tanx图象，提炼五要素（定义域、值域、周期、奇偶性、单调性）  
- **学习目标**：能脱离描点，用“关键点法”（五点作图）10秒内画出y=sinx一个周期；能从图象反推函数性质；能解释y=tanx渐近线为何在x=π/2+kπ  
- **学习内容**  
  - y=sinx：关键点(0,0),(π/2,1),(π,0),(3π/2,−1),(2π,0)；周期2π；奇函数；[-π/2+2kπ,π/2+2kπ]↑  
  - y=cosx：关键点(0,1),(π/2,0),(π,−1),(3π/2,0),(2π,1)；周期2π；偶函数；[2kπ,π+2kπ]↓  
  - y=tanx：关键点(−π/4,−1),(0,0),(π/4,1)；周期π；奇函数；渐近线x=π/2+kπ；(-π/2+kπ,π/2+kπ)↑  
  - 图象与单位圆联动：sinx纵坐标随x旋转的动态投影  
- **前置依赖**：阶段一全部内容；函数基本概念（定义域/值域/单调性）  
- **建议学习时间**：第5周·4小时（含2h动态图象手绘+2h性质速记卡制作）

### 模块2.2｜y = A sin(ωx + φ) + B 的四参数解码  
- **学习主题**：解构振幅A、角频率ω、初相φ、竖直偏移B的独立物理意义与耦合效应  
- **学习目标**：能独立分析每个参数对图象的影响；能区分“先平移后伸缩”与“先伸缩后平移”的本质差异；能准确计算相位平移量（|φ/ω|）  
- **学习内容**  
  - A：振幅（纵轴拉伸/压缩），影响值域[A·min+ B, A·max+ B]，A<0时图象翻转  
  - ω：决定周期T=2π/|ω|，影响横轴压缩/拉伸；|ω|>1 → 周期变短 → 图象变密  
  - φ：初相，“相位平移量”为−φ/ω（注意负号！），方向由−φ/ω符号决定  
  - B：竖直平移，影响平衡位置与值域中心  
  - 关键辨析：y=sin(2x+π/3) = sin[2(x+π/6)] → 先横坐标压缩为1/2，再左移π/6（非π/3！）  
- **前置依赖**：模块2.1；函数图象变换基础（平移/伸缩规则）  
- **建议学习时间**：第5–6周·6小时（含3h参数拆解实验+3h“错移量”专项辨析）

### 模块2.3｜标准变换序列的三步法定制化绘制  
- **学习主题**：建立“标准变换流程”应对任意y = A sin(ωx + φ) + B  
- **学习目标**：面对任意参数组合，能在白板上无草稿完成三步动作：① 写出标准变换顺序；② 标出关键点映射；③ 验证端点与极值  
- **学习内容**  
  - 三步法（以y=A sin(ωx+φ)+B为例）：  
    ① **标准化**：提取ω，写为y=A sin[ω(x+φ/ω)]+B  
    ② **变换链**：y=sinx  
    　→ 横坐标变为原来的1/|ω|（ω<0时加翻转）  
    　→ 左/右平移|φ/ω|（符号由−φ/ω定）  
    　→ 纵坐标变为原来的|A|倍（A<0时加翻转）  
    　→ 上/下平移|B|（B>0上移）  
    ③ **关键点映射**：原(0,0)→(−φ/ω,B)；(π/2,1)→(−φ/ω+π/(2ω),A+B)等  
  - 反向设计训练：给定“周期π、最大值5、最小值1、对称中心(π/3,3)”，求解析式 → 解得A=2,B=3,T=π→ω=2, 对称中心→φ=−2π/3 → y=2sin(2x−2π/3)+3  
- **前置依赖**：模块2.2；函数复合变换规则  
- **建议学习时间**：第6周·5小时（含2h三步法固化训练+3h反向设计实战）

### 模块2.4｜含参函数的性质分析策略  
- **学习主题**：求解含参数A,ω,φ,B的三角函数定义域、值域、单调区间、对称轴/中心、最值  
- **学习目标**：能系统处理参数符号不确定性（如ω正负影响单调性方向）；能对A=0、ω=0等退化情形做边界检验；能写出含参单调区间的规范表达式  
- **学习内容**  
  - 值域：由A,B决定 → [−|A|+B, |A|+B]（A≠0）；A=0时为常函数y=B  
  - 单调区间：先求y=sinu的单调区间u∈[−π/2+2kπ,π/2+2kπ]，再解ωx+φ∈该区间 → 注意ω符号（ω<0时不等式变号）  
  - 对称轴：sin函数在u=π/2+kπ处取最值 → ωx+φ=π/2+kπ → x=(π/2+kπ−φ)/ω  
  - 对称中心：sin函数在u=kπ处过零点 → ωx+φ=kπ → x=(kπ−φ)/ω  
  - 易错点清单：ω为负时未变号、忽略A=0退化、φ未提取导致平移量错误  
- **前置依赖**：模块2.2–2.3；不等式解法（含参数讨论）  
- **建议学习时间**：第7周·6小时（含3h参数讨论训练+3h高考真题拆解）

### 模块2.5｜y = A tan(ωx + φ) + B 的特殊性突破  
- **学习主题**：攻克正切型函数的渐近线、周期、单调性与图象特征  
- **学习目标**：能准确求出渐近线方程；能判断单调区间是否连续；能处理ω<0、A<0对图象的影响；能解决含参正切函数的定义域限制问题  
- **学习内容**  
  - 渐近线：由ωx+φ=π/2+kπ → x=(π/2+kπ−φ)/ω  
  - 周期：T=π/|ω|（非2π/|ω|！）  
  - 单调性：在每个连续区间内严格单调（A>0则↑，A<0则↓）  
  - 定义域：x≠(π/2+kπ−φ)/ω，k∈Z → 含参时需明确排除所有不满足式子的x  
  - 与正弦/余弦对比：无最大值/最小值；值域恒为R；奇偶性取决于φ（仅当φ=kπ时为奇函数）  
- **前置依赖**：模块2.1（y=tanx基础）、模块2.2（参数解码）  
- **建议学习时间**：第7周·3小时（聚焦正切特异性，避免与正弦混淆）

### 模块2.6｜阶段二闭环验证：参数敏感度与变换序贯性认证  
- **学习主题**：通过三项强制输出，完成阶段二能力微认证  
- **学习目标**：达成“无计算纸笔完成三步动作”的直觉级图象操控能力  
- **学习内容**  
  - ✅ 白板实录（拍照存档）：给定y=−3cos(4x−π/2)+1，现场完成：①标准变换序列文字描述；②原y=cosx的5个关键点映射坐标；③反向设计一个满足“周期π/2、关于点(π/4,−2)中心对称、最小值−5”的函数并验证  
  - ✅ 《参数干扰诊断表》：列举φ与ω耦合导致的3类典型错误（如“y=sin(2x+π/3)平移量误判为π/3”），每类给出单位圆/图象双视角归因  
  - ✅ 《图象变换决策检查清单》：含5条可执行指令，如“遇ωx+φ结构，必先提取ω→再算−φ/ω→最后确认ω符号对单调性影响”  
- **前置依赖**：模块2.1–2.5全部完成  
- **建议学习时间**：第8周末·3小时（集中完成三项输出，作为阶段二结业凭证）

## 阶段三：综合应用与策略元认知（第9–11周｜总54小时｜理论16h + 实践38h）

### 模块3.1｜三角恒等变换的策略选择引擎  
- **学习主题**：建立“目标导向”的恒等变形决策系统  
- **学习目标**：能根据题目目标（化简/求值/证明/解方程）自动匹配最优公式链；能识别“隐藏结构”（如1±sinx、a sinx+b cosx）并启动对应策略  
- **学习内容**  
  - 四大目标策略：  
    ▪️ **化简**：降幂（cos2α）→ 切化弦（tan→sin/cos）→ 和差化积  
    ▪️ **求值**：角配凑（75°=45°+30°）→ 特殊值代入 → 公式逆用（如cos2α=2cos²α−1→cos²α=(1+cos2α)/2）  
    ▪️ **证明**：由繁到简（从复杂侧出发）→ 两端化同（左右分别化为同一式）→ 比较法（作差/作商）  
    ▪️ **解方程**：化为单一函数（asinθ+bcosθ=Rsin(θ+φ)）→ 换元（t=sinθ）→ 讨论定义域  
  - 隐藏结构响应库：  
    ▪️ √(1±sinx) → 用半角公式，注意开方符号  
    ▪️ asinx+bcosx → 提R=√(a²+b²)，写为Rsin(x+φ)  
    ▪️ sin⁴x+cos⁴x → 用sin²x+cos²x=1降次  
- **前置依赖**：阶段一全部公式；代数变形技巧  
- **建议学习时间**：第9周·6小时（含3h策略匹配训练+3h“隐藏结构”识别题集）

### 模块3.2｜解斜三角形：定理选择决策树  
- **学习主题**：建立“已知条件→匹配定理→验证存在性→求解路径”的全流程决策机制  
- **学习目标**：能根据SSS/SAS/ASA/AAS/SSA五种已知组合，100%正确选择正弦/余弦定理；能对SSA情形完成两解/一解/无解的完整判断；能处理含参三角形存在性问题  
- **学习内容**  
  - 定理匹配表：  
    ▪️ SSS → 余弦定理求角  
    ▪️ SAS → 余弦定理求第三边  
    ▪️ ASA/AAS → 正弦定理求边（内角和先求第三角）  
    ▪️ SSA（边边角）→ 正弦定理求角 → **两解判断黄金步骤**：① 求出sinB；② 若sinB>1→无解；若sinB=1→一解（直角）；若0<sinB<1→继续③比较a与bsinA：a<bsinA→无解；a=bsinA→一解（直角）；bsinA<a<b→两解；a≥b→一解  
  - 实际应用建模：  
    ▪️ 测量问题：构造三角形→标已知边角→选定理→解出目标量→单位还原  
    ▪️ 航海问题：方位角转换为三角形内角（正北为0°，顺时针增）  
    ▪️ 力学问题：力的合成→平行四边形法则→转化为三角形求合力大小与方向  
- **前置依赖**：初中几何（三角形内角和、全等判定）；向量基础（可选）  
- **建议学习时间**：第9–10周·7小时（含4h决策树训练+3h实际应用建模）
